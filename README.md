# RepoInsight

AI-powered Repository Explanation Generator

RepoInsight analyzes software repositories and generates structured, evidence-based documentation, architecture summaries, technology stack analysis, user flows, and Mermaid diagrams using Large Language Models.

## Features

- Repository Upload
- Automated Repository Analysis
- AI-powered Documentation Generation
- Technology Stack Detection
- Architecture Summary
- Important Component Analysis
- User Flow Explanation
- Mermaid Diagram Generation
- Streaming Responses
- Download Documentation as ZIP
