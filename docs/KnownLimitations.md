# Known Limitations

- Repositories must be uploaded as ZIP archives.
- Binary files are ignored during analysis.
- Documentation quality depends on repository completeness.
- Extremely large repositories may require longer processing times.
- AI-generated Mermaid diagrams may occasionally require manual correction.
- Generated explanations are evidence-based but may include inferred information where repository evidence is incomplete.
- Private repositories must be downloaded manually before upload.
- The application currently analyzes one repository at a time.
